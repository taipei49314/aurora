# Case: iron-air-mini

Offline multi-source mini case for AURORA real-data path testing.

**Not** a historical retro study (that is Loop 3). This case only proves:

1. USPTO + jobs + news adapters compose via `merge`
2. Wire reprints do not inflate source independence
3. Package imports with zero row errors

## Reproduce

```bash
# from repo root
python -m adapters uspto adapters/fixtures/uspto_sample.json -o cases/iron-air-mini/from_uspto.json --strip
python -m adapters jobs  adapters/fixtures/jobs_sample.json  -o cases/iron-air-mini/from_jobs.json --strip
python -m adapters news  adapters/fixtures/news_sample.json  -o cases/iron-air-mini/from_news.json --strip
python -m adapters merge \
  cases/iron-air-mini/from_uspto.json \
  cases/iron-air-mini/from_jobs.json \
  cases/iron-air-mini/from_news.json \
  -o cases/iron-air-mini/package.json --strip --validate --strict

# or: make adapt-merge-demo
```
